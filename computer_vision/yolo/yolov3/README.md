# **YOLOv3 Implementation on Pytorch**
**YOLOV3** is an **object detection** model published in
the paper **<u><a href="https://arxiv.org/pdf/1804.02767">'YOLOv3: An Incremental Improvement'</a></u>** **(2018)**.
It's the third version of YOLO model family. The model has three heads allowing each one to predict different object size,
small, medium and larger. This features will be inherited by future versions.
